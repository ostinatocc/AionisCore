# Aionis Core SDK Publishing Docs

This slice adds the first direct SDK docs layer on top of the Aionis Core package work that already exists.

## What changed

1. Added an SDK quickstart.
2. Added an SDK publishing guide.
3. Updated root and package README surfaces to point at those docs.

## Why it matters

Before this step, the repository had SDK package code, examples, tests, and release verification, but the public docs surface was still too implicit.

After this change, Aionis Core has:

1. a package
2. examples
3. release verification
4. direct SDK onboarding docs

That is a much stronger baseline for external release work.
